# Typing

> 这是一个简洁打字网站，可以用来记录打字速度，以及打字错误率。支持登录，记录打字数据，也会以此开发更多有趣的功能，敬请期待。

整个项目包含前端和后端，前端使用 Vue3 + Vite，后端使用 Nodejs + Koa2 + Leancloud。该项目是前端部分，后端也将在未来开源。
